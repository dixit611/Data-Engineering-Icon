Data Engineering Overview
This project brings together key data engineering components to support a clean, scalable and reliable analytics setup.

Fabric
Microsoft Fabric provides a unified space for data pipelines, lakehouse storage and analytics. It helps keep workflows simple and consistent.

Medallion Architecture
The Medallion structure separates data into Bronze, Silver and Gold layers. Bronze holds raw data, Silver applies cleaning and transformations and Gold prepares data for reporting.

Power BI
Power BI connects to curated data and provides dashboards and reports that are easy to explore and share.

Kafka
Apache Kafka handles real-time streaming, making it possible to process and move data with low delay.
